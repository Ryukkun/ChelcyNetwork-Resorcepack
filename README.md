# &nbsp;　ChelcyNetwork-Resorcepack
アスレチックサーバー「[ChelcyNetwork](https://www.mchel.net/)」でのみ使えるリソースパックです。  
アスレ選択画面で、どにどのアスレがあるかが わかりやすくなります。<br>

## ◈ 注意
・必ずoptifineを導入し、品質設定のカスタムアイテムをオンにしてください。
